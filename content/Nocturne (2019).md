# Cruel & Curious: Nocturne — 2019

| | |
|--|--|
| **Title** | Cruel & Curious: Nocturne |
| **Year** | 2019 |
| **Theme** | Night / darkness |
| **Venue** | [[Stowe Barton]], North Cornwall |
| **Dates** | 27–28 September 2019 |
| **Artists** | 35+ |
| **Performance** | Ezmay Grace (Saturday 1:30–3pm) |

## Brief

The Nocturne theme extended the exhibition's uncanny atmosphere into darkness — figures moving through night landscapes unaware of what surrounds them.
